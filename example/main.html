PubKeys Notes:
Request pubkeys available for multiple accounts in one call by comma-separating chain:username pairs.
If required for authentication, app/server must still verify that public keys match those on chain.
Supply chain short-code only with colon (ie. "eos:") to popup a prompt for user to select chain account.
Supply object with { account, active, posting, memo } to add/update private keys for a WhaleVault account.

Sign Buffer



SignBuffer Notes:
If message is a STEEM-compatible "operations" object, the tx will be signed and sig returned in hex format.
If message is an EOS-compatible "actions" object, tx will be signed and broadcast.
If message is raw format, message will be signed as a 'binary' byte buffer and sig returned in hex format.
If message is a jsonrpc object including url (K defaults to "steem_jsonrpc_auth"),
     a signedCall-style jsonRpc message will be signed and broadcast.
